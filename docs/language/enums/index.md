| Enum | BLF type | Values | Used by |
|------|----------|--------|---------|
| [Sounds](/language/enums/sounds) | `e_megalo_sound` | 240 | `play_sound`, `hud_post_message` |
| [Math operations](/language/enums/math-operations) | `e_math_operation` | 13 | `set`, `set_score`, `object_adjust_*`, … |
