# Actions

Megalo **actions** are imperative statements inside triggers. Each page documents one `e_action_type` opcode: what it does, a real invocation example, and which Reach builds support it.

For syntax rules (operands, `set`, `for_each`, and action families), see [action](/language/elements/trigger/action).
