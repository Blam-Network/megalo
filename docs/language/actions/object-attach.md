# object_attach


<AvailabilityCard reach="yes" />

## Description

Attaches one object to another with offset and constraint parameters.

<ActionParameters />

## Example

```megalo
action object_attach the_flag current_team.goal 0 0 0.15
```

Example from HREK `broken/1Flag_Boneyard.txt`.
