# timer_set_rate


<AvailabilityCard reach="yes" />

## Description

Sets a timer's tick rate. `0` pauses, `-1` counts down, `1` is normal speed.

<ActionParameters />

## Example

```megalo
action timer_set_rate period_timer 1
```

Example from HREK `broken/1Flag_Boneyard_Extreme.txt`.
