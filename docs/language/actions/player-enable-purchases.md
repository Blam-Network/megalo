# player_enable_purchases


<AvailabilityCard reach="yes" />

## Description

Enables or disables requisition purchases by category and alive/dead state.

<DocsBlock type="note" title="Requisition System">

`player_enable_purchases` is part of the Requisition System that was scrapped before Halo: Reach's release. While much of the requisition functionality remains in the game to this day, the requisition menu is long gone, rendering this action effectively useless outside of early pre-release builds.

</DocsBlock>

<ActionParameters />

## Example

```megalo
action player_enable_purchases current_player dead all true
```

Example from HREK `broken/dmiller_sve.txt`.
