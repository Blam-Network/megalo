# player_death_get_damage_type


<AvailabilityCard reach="yes" />

## Description

Writes the damage type from a death event into an out-variable.

<ActionParameters />

## Example

```megalo
action player_death_get_damage_type dead_player damage_id
```

Example from HREK `broken/1Flag_Boneyard.txt`.
