# team_set_coop_spawning


<AvailabilityCard reach="yes" />

## Description

Enables or disables cooperative spawning for a team.

<ActionParameters />

## Example

```megalo
action team_set_coop_spawning current_team true
```

Example from HREK `broken/1Flag_Boneyard.txt`.
