# apply_player_traits


<AvailabilityCard reach="yes" />

## Description

Applies a named `player_traits` block to a player.

<ActionParameters />

## Example

```megalo
action apply_player_traits current_player uber_traits
```

Example from HREK `broken/derekball.txt`.
