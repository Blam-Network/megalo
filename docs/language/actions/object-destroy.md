# object_destroy


<AvailabilityCard reach="yes" />

## Description

Destroys an object immediately (distinct from `delete_object` in engine handling).

<ActionParameters />

## Example

```megalo
action object_destroy ball
```

Example from HREK `broken/hot_potato.txt`.
