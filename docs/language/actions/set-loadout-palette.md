# set_loadout_palette


<AvailabilityCard reach="yes" />

## Description

Assigns a loadout palette tier to a player or team.

<ActionParameters />

## Example

```megalo
action set_loadout_palette player current_player slayer_loadouts
```

Example from HREK `broken/equipment_slayer.txt`.
