# object_get_distance


<AvailabilityCard reach="yes" />

## Description

Writes the distance between two objects into an out-variable.

<ActionParameters />

## Example

```megalo
action object_get_distance current_object current_player distance_to_player
```

Example from HREK `broken/slayer-revenant.txt`.
