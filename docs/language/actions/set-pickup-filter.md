# set_pickup_filter


<AvailabilityCard reach="yes" />

## Description

Restricts who can pick up an object (`all`, `allies`, `enemies`, `no_one`).

<ActionParameters />

## Example

```megalo
action set_pickup_filter flag enemies
```

Example from HREK `broken/calvinball.txt`.
