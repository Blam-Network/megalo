# player_adjust_money


<AvailabilityCard reach="yes" />

## Description

Modifies a player's requisition money with a [math operation](/language/enums/math-operations).

<ActionParameters />

## Example

```megalo
action player_adjust_money current_player add 5
```

Example from HREK `broken/1Flag_Boneyard.txt`.
