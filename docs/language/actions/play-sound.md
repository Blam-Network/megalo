# play_sound


<AvailabilityCard reach="yes" />

## Description

Plays a sound for a player, team, or everyone. Sound tokens are listed in [Sounds](/language/enums/sounds).

<ActionParameters />

## Example

```megalo
action play_sound everyone ctf
```

Example from HREK `broken/1Flag_Boneyard.txt`.
